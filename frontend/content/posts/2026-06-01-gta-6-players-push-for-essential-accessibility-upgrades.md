# GTA 6: Players Push for Essential Accessibility Upgrades

The dedicated GTA 6 community is already looking ahead to the highly anticipated title, with a recent Reddit thread sparking a vital conversation about accessibility. Initiated by user /u/Dream_SMP_4_Life, the discussion on r/GTA6 underscores a strong desire for Rockstar Games to implement more comprehensive accessibility options, ensuring the game is enjoyable and playable for a broader audience. The thread highlights several specific features that players believe would significantly enhance the user experience, emphasizing the growing importance of inclusivity in modern gaming.

## Key Takeaways
- Customizable subtitles (including options for color, background, and size)
- Customizable waypoint colors (at least offering a selection of colors)
- Improved user interface for displaying in-game money
- Return of the GPS voice feature, similar to its implementation in GTA 4
